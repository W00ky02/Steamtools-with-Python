# Steamtools-with-Python
I've created Steamtools with Python, which isn't Spyware.

# Tutorial:

1. Click on "Generator", paste the AppID and download the Manifest and Lua files.
2. Click on "Manifest and Lua" and drag the files in.
3. Restart Steam

It shuold show the Game in your library.

# Uninstall:

1. Click on "Uninstall" and all Games with Manifest and Lua files should show.
2. Select the Files and Click "Uninstall".
3. Restart Steam

The Game should be gone.
